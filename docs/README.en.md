# ShipLua documentation

The project's canonical documentation remains in Brazilian Portuguese. Each
document in Portuguese has a paired English version in the same directory,
with the suffix `.en.md` before the extension.

Examples:

- `PLAN.md` → `PLAN.en.md`;
- `docs/api/schema-contract.md` → `docs/api/schema-contract.en.md`;
- `rfcs/0001-shiplua-architecture.md` → `rfcs/0001-shiplua-architecture.en.md`.

The English versions preserve code, commands, paths, identifiers and
technical contracts. In case of discrepancy, consult the document in Portuguese,
which remains the canonical source of the project record.
